# 托业800+ 高频词组 - 闪卡背单词

翻转卡片式 TOEIC 单词学习网页应用，收录 800+ 高频词汇。

## 功能

- 闪卡翻转：点击卡片查看词义和例句
- 语音朗读：点击喇叭听发音（Google TTS + SpeechSynthesis 双引擎）
- 认识/不认识标记，自动记忆进度
- 随机洗牌、上下翻页
- 键盘快捷键：← → 翻页 | 空格 翻转 | S 朗读 | K 认识 | U 不认识
- 进度追踪 + 统计展示

## 使用方式

直接用浏览器打开 `index.html` 即可，无需安装任何依赖。

### 在线部署

**surge.sh（命令行，永久免费）**
```bash
npx surge .
```

**Netlify Drop（浏览器拖拽）**
打开 https://app.netlify.com/drop ，拖入整个文件夹。
